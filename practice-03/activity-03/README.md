# Activity 03: SPI BMP280 Sensor

Place the ESP32 Arduino sketch for the SPI sensor exercise here.

Expected implementation:

* communicate with a BMP280 sensor through SPI, or with another project-relevant SPI device
* acquire barometric pressure
* acquire temperature
* display the measured values through the Serial Monitor or an LCD screen

Suggested sketch name:

* `spi-bmp280-monitor.ino`
