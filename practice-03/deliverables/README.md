# Deliverables

This directory contains the evidence and presentation material requested in the Practice 03 guide.

Expected contents:

* `presentation/`: slides or exported presentation files for the final submission
* `videos/`: one subdirectory per implementation exercise with the corresponding local video file, public link when available, subtitles, and supporting notes

Before submitting:

* verify that the presentation summarizes the six requested points
* verify that the presentation includes the research component and conclusions
* verify that each of the first four points has working demonstration video evidence
* verify that every shared link has public access enabled
* verify that the notes match what is actually demonstrated
